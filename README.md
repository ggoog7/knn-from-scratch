# KNN (K-Nearest Neighbors)

Реализация алгоритма **KNN (K-ближайших соседей)**  на Python.  
Алгоритм предсказывает, пойдёт ли человек гулять, основываясь на температуре и уровне осадков.

##  Что делает проект
- Предсказывает класс (0 или 1) для новой точки на основе **k ближайших соседей**
- Использует **Евклидово расстояние** для поиска соседей
- Визуализирует данные и результат на графике

##  Как работает KNN
1. Алгоритм запоминает все данные (не учится!)
2. Для новой точки считает расстояние до всех известных точек
3. Выбирает k ближайших соседей
4. Голосует: какой класс встречается чаще — тот и ответ

##  Данные
Признаки:
- Температура (°C)
- Осадки (мм)

Метки:
- 1 = "Пойду гулять"
- 0 = "Не пойду гулять"

##  Результат
Новая точка: температура 15°C, осадки 0 мм
Предсказание: Пойду гулять!
